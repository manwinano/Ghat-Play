# SFML 3.0.2 Chat

Клиент-серверное приложение для обмена сообщениями в локальной сети.

## Описание

Программа состоит из двух частей:
- **Сервер** — принимает подключения клиентов и пересылает сообщения между ними
- **Клиент** — подключается к серверу и отправляет/получает сообщения

## Как работает

**Сервер:**
- Запускается и ожидает подключения на порту 53000
- При подключении нового клиента отправляет приветствие и уведомляет остальных
- Получает сообщения от клиентов и рассылает их всем остальным
- Отслеживает отключения и уведомляет об этом

**Клиент:**
- Подключается к серверу по IP-адресу и порту
- Отправляет сообщения с указанием никнейма
- Получает сообщения от других клиентов в реальном времени
- Поддерживает команды: /quit (выход), /help (справка)

## Требования

- C++17
- SFML 3.0.2 (Network, System)
- Компилятор: Visual Studio 2019/2022, GCC 9+, Clang 10+

## Установка SFML

### Windows
1. Скачайте SFML 3.0.2 с [официального сайта](https://www.sfml-dev.org/download.php)
2. Распакуйте в `C:\SFML`
3. Добавьте в проект:
   - Include: `C:\SFML\include`
   - Lib: `C:\SFML\lib`
   - DLL из `C:\SFML\bin` в папку с .exe
